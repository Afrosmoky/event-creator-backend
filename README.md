## Event Creator – Backend API

Overview

This project is a Laravel-based REST API for a hall and seating management application.

It supports dynamic event room configuration, table management, and guest assignment logic.

The system is designed to handle structured event layouts and role-based operations through clean backend logic.

⸻

## Use Case

The application allows event organizers to:  
•	Configure event hall layout  
•	Define tables with size and parameters  
•	Add elements (stage, DJ area, buffet, etc.)  
•	Assign guests manually or in groups  
•	Manage seating arrangements dynamically

⸻

## Architecture
•	Laravel 11  
•	REST API architecture  
•	Clean service layer  
•	Structured validation  
•	Separation between domain logic and delivery layer

The backend is designed to remain frontend-agnostic.
